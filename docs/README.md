# docs/

Drop project media and write-ups here.

- **`Video-Project.gif`** — the screen recording embedded in the root README's
  "See it in action" section. A GIF autoplays inline on GitHub; if you'd rather use
  an MP4 (better quality), it needs to be linked instead (upload via a GitHub Release
  or by dragging it into a README edit on github.com).
- **`LINKEDIN.md`** — ready-to-post LinkedIn article + short feed post for the launch.
- **screenshots** — any stills you want to embed in the README.
